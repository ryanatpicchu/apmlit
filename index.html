About Amplit

Amplit is a personal project built by an independent developer to manage their own social media presence more efficiently. It is not a commercial product, not sold as a service, and does not process data on behalf of third-party clients.

The tool combines AI-assisted drafting, scheduled publishing, and engagement analytics into one local-first workflow. It uses the Threads API and Instagram Graph API to operate on the developer's own accounts.

What it does

Trend discovery

Gathers public trending topics from Google Trends, Dcard, and the Threads keyword search API to inform content direction.

Draft generation

Uses Anthropic Claude to draft posts that match a configured persona, tone, and industry context.

Scheduling

Schedules approved posts for automated publishing to the developer's own Threads and Instagram accounts.

Engagement analytics

Retrieves performance metrics (views, likes, replies) for the developer's own posts to refine content strategy.

Reply management

Reads replies on the developer's own posts and drafts reply suggestions for manual approval before sending.

Feedback loop

Identifies high-performing posts and drafts follow-up content based on the patterns that drove engagement.

How Amplit uses platform data

Amplit only accesses data belonging to the developer's own Threads and Instagram accounts, plus public trending keywords. Below is a summary of what is retrieved and why.

  • Own posts & insights
    Retrieved via Threads / Instagram API to display and analyze performance in the developer's private dashboard. Not shared externally.
  • Own replies & mentions
    Retrieved for reply management. Reply content is shown only in the developer's private dashboard. Never stored beyond what's needed for display.
  • Public keyword search
    Public Threads posts matching a keyword are displayed for trend research. No data is stored persistently; results are fetched on demand.
  • Follower counts
    Aggregate number only, used to plot the developer's own growth trend over time. No individual follower identities are collected.
  • AI processing
    Post drafts are generated via Anthropic Claude. Only the developer's brand settings and selected trending topics are sent. No follower PII is ever sent to the AI.
  • Data location
    All data is stored in a private PostgreSQL database on the developer's own infrastructure. Media assets are stored in Cloudflare R2 with access restricted to the developer.
  • Not collected
    Amplit does not collect, sell, or share data about any third-party users, followers, or other accounts beyond what is publicly available through official APIs.
